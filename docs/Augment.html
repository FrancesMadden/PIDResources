Augment with existing identifiers

  1. Aspects

Externally-created persistent identifiers are included for items and entities within metadata


Explanation

PIDs such as Wikidata IDs, ISNIs, Getty URIs, DOIs, Gazetteer URIs and IDs from other collections can be added to collection management systems. These can be included for related collection items, an identifier for a collection item itself where it is represented in another identifier scheme and authority files.

Benefits

  • Links with other collections.
  • Reduces workload of creating institution specific authority files.

Dependencies

Baseline required.

Examples


Compatible PID Types

  • Wikidata IDs
  • Getty URIs
  • Gazetteer URIs
  • Digital Object Identifiers from other collections
  • Handles from other collections
  • ARKs from other collections
  • URIs from other collections

For more compatible identifiers, see Which persistent identifier does what?
